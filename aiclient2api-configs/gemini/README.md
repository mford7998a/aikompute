# Place your Gemini OAuth credential files here.
# 
# How to get them:
# 1. Open AIClient-2-API Web UI at http://localhost:3001
# 2. Go to "Provider Pools" → Gemini → "Generate Authorization"
# 3. Complete Google OAuth login
# 4. The credential file is auto-saved here
#
# Or manually:
# 1. Install Gemini CLI: npm install -g @anthropic-ai/gemini-cli
# 2. Run: gemini (this opens OAuth flow)
# 3. Copy ~/.gemini/oauth_creds.json to this directory
#
# Name files: oauth_creds_1.json, oauth_creds_2.json, etc.
# Each file = one account in the rotation pool.
