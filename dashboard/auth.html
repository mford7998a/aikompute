Chat Completion

POST /v1/chat/completions

{
  "model": "claude-sonnet-4-6",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "Explain quantum computing"}
  ],
  "temperature": 0.7,
  "max_tokens": 1000,
  "stream": false
}

Python SDK Example

from openai import OpenAI

client = OpenAI(
    api_key="sk-inf-your-key-here",
    base_url="/v1"
)

response = client.chat.completions.create(
    model="gemini-3.0-pro",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)

Service Reliability

Our gateway uses a multi-layered infrastructure to ensure 99.9% availability:

  1. Requests are balanced across a distributed pool of processing nodes.
  2. If a node becomes busy or goes offline, our system instantly redirects your request to a healthy alternative.
  3. Integrated load-bearing ensures that even during high-traffic periods, your API calls remain responsive.
  4. Periodic health checks proactively monitor all nodes to guarantee consistent performance.

This architecture provides a seamless experience without manual endpoint management.
